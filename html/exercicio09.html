Este tutorial é o ponto de partida para desenvolvimento de layouts CSS. Se você prentende projetar
uma página, um documeto ou um site inteiro baseado em CSS e sem uso de tabelas complexas o perfeito entendimento deste tutorial vai lhe fornecer os conceitos básicos de posicionamento CSS.

Este assunto iniciará você nas técnicas avançadas de
posicionamento CSS. Para seu perfeito entendimento é
necessário que você domine com destreza o uso das
propriedades do CSS.
